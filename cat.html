貓咪

瞇瞇眼 大胖橘

這隻貓咪是在麻六甲聖保羅山拍攝的
這隻貓咪會被叫瞇瞇眼的原因:
  1. 想睡覺了，太刺眼了
  2. 身體不好
  3. 很滿意現狀
  4. 毫無敵意

這隻貓咪是在猴硐貓村拍攝的
這隻貓咪會被叫大胖橘的原因:
  1. 胖橘形象出自《加菲貓》電影裡的加菲貓
  2. 貪吃懶惰，但是又可愛
  3. 圓圓腦袋
  4. 圓圓眼睛
圓滾滾 小釘鐺

這隻貓咪是在哈爾濱東正教堂拍攝拍攝的
這隻貓咪會被叫圓滾滾的原因:
  1. 很有威嚴
  2. 有如懷孕
  3. 野生東北豹
  4. 圓滾滾的

這隻貓咪是在桃園某火鍋店拍攝的
這隻貓咪會被叫小釘鐺的原因:
  1. 穿著圍裙的可愛貓咪造型
  2. 逗趣的聲音表情
  3. 互動點播唱恭喜發財
  4. 體驗新型態科技的用餐服務
